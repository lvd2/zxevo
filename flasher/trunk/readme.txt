Прошивальщик Flash-ROM
======================

v100103 (DDp)
~~~~~~~
Реализована концепция аварийного программатора Flash-ROM
в виде самостоятельной конфигурации.

Сначала в основную память AVR загружается флэшер.
При старте он ищет на SD файл ZXEVO.ROM.
Если всё в порядке просит пользователя нажать SoftReset,
полностью стирает Flash-ROM и пишет данные из файла.
Размер файла может быть от 1 байта до 512Кб.
Пишется блоками по 512 байт. Если размер файла не кратен 512,
пишется мусор из сектора (то, что прочиталось с SD).

Сейчас не анализируются ошибки flash-чипа
и не делаются никакие проверки записанного.

В RS-232 комментируются действия (115200-N-2).
На видео сигнала нет.

На данный момент для базовой конфигурации
собирать ZXEVO.ROM нужно в следующем порядке:
0 - ServiceROM (GLUK)
1 - TR-DOS
2 - Basic128K
3 - Basic48K
