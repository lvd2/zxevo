
Исходники взяты отсюда http://john.ccac.rwth-aachen.de:8000/ftp/as/source/c_version/asl-current.tar.gz

Изменения:
- перед сборкой для Win32 копируется файл Makefile.def в папку исходников.
- перед сборкой в папку исходников копируется make.exe
- файл codez80.c в функции CodeEnd() строка "else Type = UInt16;" изменена на "else Type = UInt32;"
  для возможности собирать бинарники с размером больше 64к.
- файл datatypes.h увеличен STRINGSIZE до 1024 байт (длина строки исходного текста).

Как сделать в makefile не разобрался, потому сделал через батник.
Использовался gcc 8.1.0 и make 3.81 (более новая make версия собирать отказывается из-за ошибок).
